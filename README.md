En este repositorio se ha creado una visualización, a partir de la base de datos de [Playa de Canarias](https://opendata.esri.es/datasets/84ddbc8cf4104a579d579f6441fcaa8a_0/explore?filters=eyJDb211bmlkYWRfIjpbIkNhbmFyaWFzIl19&location=32.757017%2C-12.409024%2C6&showTable=true), respondiendo a preguntas o clasificando estas según sus características. 

##### Descripción de archivos
- index.html: resultado final de la visualización tipo storytelling.
- limpiezadedatos.ipynb: fichero para la preparación de los datos y análisis de los mismos.
- Playas_canarias.csv: dataset original
- Playa_canarias_limpio.csv: dataset filtrado, limpio y preparado para la visualización.
